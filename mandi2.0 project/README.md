# Mandi-Agent 2.0 | A2A Market Protocol

Mandi-Agent 2.0 is an autonomous Agent-to-Agent (A2A) negotiation platform built for the GDG Hackathon. It streamlines agricultural trade by allowing AI agents to handle price discovery, quality verification, and logistics recovery.

## 🚀 Live Demo Features
- **Autonomous A2A Negotiation**: AI agents discover prices and settle deals.
- **God Mode Logistics**: Real-time road-block simulation and autonomous rerouting.
- **AI Strategic Insights**: Real-time analysis of agent tactics.
- **Market Pulse Ticker**: Live commodity price tracking.
- **A2A Topology**: Real-time network traffic visualization.

## 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), JavaScript (ES6+)
- **Icons**: Font Awesome 6.4
- **Fonts**: Google Fonts (Outfit)

## 📦 Deployment
This project is a "no-build" Vanilla JS application. You can deploy it instantly using:

### Option 1: Vercel (Recommended)
Run the following command in the project root:
```bash
npx vercel
```

### Option 2: Netlify Drop
Drag and drop the project folder to [Netlify Drop](https://app.netlify.com/drop).

### Option 3: GitHub Pages
1. Initialize a Git repo.
2. Push to GitHub.
3. Enable GitHub Pages in Repository Settings.
